MoodMusic – Music Recommendation System

MoodMusic is a hybrid music recommendation system that suggests songs based on a user’s mood and listening preferences. It is built using concepts from Andrew Ng’s Machine Learning course. The system combines content-based filtering and collaborative filtering implemented from scratch. It uses data from Spotify audio features such as valence, energy, and danceability to represent songs and identify patterns related to mood. At the same time, collaborative filtering with matrix factorization learns user preferences from interaction data. By integrating both approaches, the system generates more personalized and accurate recommendations of music. This demonstrates how core machine learning techniques can be applied to solve real-world problems.
